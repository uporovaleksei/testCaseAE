## Установка и запуск Frontend

В новом терминале:

 1. cd frontend
 2. npm install
 3. npm run dev

## Установка и запуск Backend

 1. cd backend
 2. npm install
 3. npm run dev

**( ! )** Для frontend необходимо создать файл окружения .env /frontend/.env с переменной VITE_API_URL=http://localhost:3000/api
